Expand description

Generic wrapper for any container type.

This is implemented over standard Rust containers (Vec, boxed slices, etc.) in addition to more exotic types. &[T] is insufficiently generic for types that return a ref handle to T, instead of &T directly, so this allows for more complex container types.

Methods are prepended with c_* to avoid namespace conflicts.
